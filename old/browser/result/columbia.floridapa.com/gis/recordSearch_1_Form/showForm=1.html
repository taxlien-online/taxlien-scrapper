Search Tips:
The above form will create a custom search based on ALL the input that you enter. If you search on more than one field, your results will be limited to only those matching records. For example; Entering "SMITH" in the Name field, and selecting "VACANT" as the Land Use type will retrieve all VACANT parcels owned by somebody named SMITH.

Owner's Name Search: format: (LastName FirstName) ... For best results, enter only first part of name.
- "Smith J" will return all SMITH's with the first name starting with J
Address Search: Enter as much or as little of the address as you wish. Leave off the Street Number & Street Type if you're not sure.
Parcel ID Search: (Parcel Identification Number) Enter as much of the PIN as you know. (you can add dashes "-" on your input for easy reading!)
Geographic Search: Section/Township/Range: Use these to narrow the search results to particular geographic location.
Land Use, Acreage (min/max): Select a Land Use type to limit the amount of records retrieved. Entering a Min. and Max. Acreage value will narrow results even further.
Structure Search: Heated Square Footage: Heated Square Footage is also known as Base Area.
Year Built: Enter a 4 digit year.
Sales History Search: Sale Date (min/max): format: (Month / Day / Year , Month / Year) Specify a Date Range to find parcels that have been sold between the dates entered.
Sale Price (min/max): Enter a Sale Price Range. (TIP: Enter a Sale Price GT >= $101 to ignore all Trust or non arm length sale transactions)
Vacant or Improved: Select to filter Sales hits on Vacant or Improved Sales
Qualified Sales: Select to filter Sales hits on Qualified Only Sales
Last Sales only: Select this option to only Search the Last Sale for each property. (unchecked will search all recorded sales)
Sale Book/Page: Enter the Official Records Book and Page number of the recorded Deed to find the associated parcel.
